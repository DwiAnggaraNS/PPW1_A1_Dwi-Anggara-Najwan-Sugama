    Analisis dan Prediksi Volume Produksi Sampah di Provinsi Daerah Istimewa Yogyakarta Menggunakan Model Machine Learning: Suatu Langkah Proaktif dalam Pengelolaan Sampah
  • Deskripsi Proyek:

    Projek ini dimulai dengan pengumpulan data yang bersumber dari Badan Perencanaan Pembangunan Daerah (Bappeda) DIY, diikuti dengan tahap pembersihan data untuk menangani missing values dan outliers guna memastikan keakuratan data. Dilanjutkan dengan analisis eksploratif data (EDA) untuk menemukan pola dan tren dalam volume produksi sampah. Selanjutnya, dilakukan pemodelan Machine Learning yang memanfaatkan faktor-faktor kunci seperti jumlah Titik Pengumpulan Sampah (TPS), kapasitas Tempat Pembuangan Sampah Terakhir (TPST) regional, total unit TPST, dan Volume Sampah yang Dikelola per hari. Proses akhir melibatkan evaluasi model menggunakan cost function untuk menentukan model terbaik, dengan harapan hasil proyek ini dapat memberikan kontribusi berharga bagi Pemerintah Daerah Istimewa Yogyakarta dalam mengambil langkah-langkah proaktif dalam pengelolaan sampah.
